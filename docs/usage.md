# Usage

## Open the tool
Open `index.html` in a modern browser.

## Basic workflow
1. Select icon style
2. Set label and appearance
3. Adjust accent color
4. Preview result
5. Export to SVG or PNG

## Recommended use
Use exported icons for tiles, quick links, navigation elements, or internal UI mockups.

## Notes
Verify exported icons for:
- centering
- correct background shape
- correct accent color
- transparency behavior
